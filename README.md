# alipay_wx_pay
微信支付宝个人收款系统，个人免签收款解决方案。完全免费 资金即时直接到达本人账号， 支持支付宝、 微信，无需备案 无需签约，无需营业执照身份证。


运行环境： centos、nodejs、nginx。
简介：支持个人网站、安卓App、微信公众号、Pc软件的个人收款接入，资金实时直接到账您的支付宝/微信余额中，不经过中间平台，无手续费。支付宝无需上传收款二维码，支持H5唤醒支付，支持回调通知。真正的完全开放源代码，支持php/java/python等语言直接接入。
